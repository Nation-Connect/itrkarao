Import-Export Code(IEC) Registration


IEC Code is unique code provided to every Importer and Exporter. IEC code is a 10 Digit Code. IEC Code is must for every business that is in import and export. IE Code stands for Import Export Code. It is required for any person importing or exporting goods and services to or from India.

IEC is issued by the Directorate General of Foreign Trade (DGFT), Ministry of Commerce and Industries, Government of India. Importers are not allowed to proceed without this code and exporters can’t take benefit of exports from DGFT, customs, Export Promotion Council, if they don’t have this code.

Why IE CODE Required?

  1. Importers who import goods into India must quote IE Code while clearing customs. Also, banks require the importers IE Code while sending money abroad.
  2. Exporters who export goods or services from India must quote IE Code while sending shipments. And banks require the exporters IE Code while receiving money from abroad.
  3. Filing of any return is not required if u have IE Code. Once, an IE Code is issued there are no further procedures required to maintain validity of the IE Code.There are no filings required to DGFT ,even if import or export transactions occur .
  4. It is not necessary to incorporate a business entity for obtaining IE Code.Even individuals who are proprietors of a business can obtain IE Code in their name.
  5. IE Code can easily be obtained from DGFT within 10-15 working days after submission of the application along with all the necessary information.
  6. IEC alloted to an applicant is valid for ever and no need to renew the same.
