GST Return

 

It is a document containing details of income which a taxpayer is required to file with the tax administrative authorities. This is used by tax authorities to calculate tax liability.A registered dealer has to file GST returns under GST that include:

 

  • Purchases
  • Sales
  • Output GST (When you sale something)
  • Input tax credit (GST paid when you purchase something)

Every calendar month registered person paying GST is required to furnish an electronic return .

The tax paid by a registered dealer depends upon the income declared in the tax return filed with the tax authorities.

As per the current system of GST Return filing requires a taxpayer to update outward supplies information in GSTR 1. And then file return in GSTR 3B.

From April 1, 2019, the government is planning to implement the new GST Return design. This require taxpayers having an annual turnover of over Rs 5 Crores to file monthly return only. Thus,business owners,having an annual turnover of upto Rs 5 Crores would have the option to file return Quarterly..

GSTR – 1: Return for Outward Supplies

This return requires the sales transactions of a business in a particular month.

Who Needs To File GSTR-1?

Every normal registered taxpayer required to file GSTR-1 each month. This require:

  1. Credit notes
  2. Revised invoices issued pertaining to your outward supplies.
  3. Invoices
  4. Debit notes.
Due Date of Filing GSTR-1

For filling such return the standard date for filing GSTR-1 is 10 days from the end of the month . However, the due date to file GSTR 1 can be extended by the commissioner.

GSTR – 3B: Summary of Inward and Outward Supplies

GSTR 3B is a self declaration showcasing the summary of GST liabilities of the taxpayer for the tax period . Moreover, it helps timley discharge of tax liablities of the tax payer.

GSTR-3B is a form that cannot be revised. Furthermore, the compliance of comparing invoices between supplier and purchaser does not require. Thus both the suppliers and the recipients file the GSTR-3B form separately.

Who Needs To File GSTR-3B?

GSTR-3B is required to be file by registered taxpayer . It is also filed during the tax periods for which the tax liability is zero.

Due Date for Filing GSTR-3B

By the 20th of the month succeeding the GSTR-3B must be filed. The registered person needs to file a NIL return for that period if there is no transaction in that particular month.
